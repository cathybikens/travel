Responsive Design

Responsive design is the practice of creating websites that adapt to fit any screen size, from tiny mobile phones to large desktop monitors. CSS plays a crucial role in achieving this flexibility.

Core concepts

  • Relative units e.g. % , rem, em, vh, vw
  • Flexible layouts(css flex and grid)
  • Media Queries- Conditional statements that apply specific styles based on screen size, orientation, and other factors.
